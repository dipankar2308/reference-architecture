# Hybrid Low-Code/Pro-Code Template

## Pattern

Power App -> Custom Azure Function API -> .NET Service -> Cosmos DB

## Includes

- Power Apps canvas app source (JSON export)
- Custom connector for Azure Functions
- .NET API with Clean Architecture
- ARM deployment templates

## Use Case

Field-facing apps with admin backoffice + reporting

## Architecture Benefits

- Low-code for rapid UI iteration
- Pro-code for complex business logic
- Shared data model
