# Reference Architectures
Enterprise-grade starter templates showcasing modern architecture patterns with Azure and Power Platform.

## Templates Included

- Clean Architecture + Hexagonal (.NET Core + Azure)
- Event-Driven Microservices (Azure Service Bus + AKS)
- Hybrid Low-Code/Pro-Code (Power Platform + Custom APIs)

Each template includes:

- Architecture Decision Records (ADRs)
- C4 diagrams
- Azure ARM templates
- Security-hardened plugins (AEGIS-inspired)
- GitHub Copilot instructions

## Quick Start
```bash
git clone https://github.com/[your-username]/reference-architectures
cd clean-hexagonal
dotnet restore
```

## Architecture Patterns Covered

- Clean Architecture
- Hexagonal (Ports & Adapters)
- Event-Driven Microservices
- Modular Monoliths
- Resilience Patterns
